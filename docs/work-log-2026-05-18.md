## Mobile UX Improvements Shipped

The shared text-tool UI is primarily implemented in:

- `assets/js/text-tools.js`
- `assets/css/text-tools.css`

These improvements are now live:

- Bottom mobile draft composer behaves like a compact sheet.
- Draft sheet can expand/collapse.
- Page behind the draft sheet remains scrollable while the draft is open.
- Draft sheet height was reduced from a large panel to a compact editor.
- Copy and clear buttons were moved into the draft editor area on mobile.
- Character count appears inside the editor.
- Recent history area is more compact.
- Draft sheet now has a clearly distinct light blue-gray background.
- Draft editor input remains white for contrast.
- The sheet has border and shadow to make the layer relationship clear.
- Copy action gives visible feedback: button text changes to `コピー済み`, toast shows `コピーしました`.
- Double-tap zoom prevention remains active to reduce accidental zoom on mobile.
