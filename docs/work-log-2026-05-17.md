## Mobile Testing Workflow

Created a lightweight mobile regression workflow instead of installing a full simulator.

Command:

```bash

```

Files:

- `scripts/mobile-audit.mjs`
- `docs/mobile-testing.md`
- `.gitignore` ignores `reports/mobile-audit/`

The audit:

- Starts a local static server.
- Launches local Chrome headless.
- Uses Chrome DevTools Protocol for mobile emulation.
- Tests:
  - iPhone SE: `375x667`
  - iPhone 15: `390x844`
  - Pixel: `412x915`
