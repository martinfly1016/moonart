📖 総合概要

月文字とは？

月文字（つきもじ）は、月の満欠を表現する絵文字🌑🌒🌓🌔🌕🌖🌗🌘を使って作られる独特なテキストアートです。本ツールは、普通のテキストや画像を美しい月文字アートに変換する革新的なWebアプリケーションです。

本ツールの価値

  • 芸術的表現: 従来のテキストアートを超越した、月の神秘的な美しさを活用した新しい表現手法
  • 多様な用途: SNS投稿、メッセージカード、デジタルアート作品、プレゼンテーション等に活用可能
  • 無料で簡単: インストール不要、ブラウザ上で即座に利用開始
  • 高度な技術: 垂直化処理、横向加強算法など独自の最適化技術を搭載
  • 完全オリジナル: 世界初の本格的な月文字生成システム

🚀 基本操作ガイド

Step 1: 入力方式の選択

  1. テキスト入力タブ: 文字から月文字を生成
  2. 画像入力タブ: 画像から月文字を生成

Step 2: テキスト入力の場合

  1. 「テキスト入力」タブを選択
  2. 入力フィールドに変換したいテキストを入力（例：「月文字」）
  3. 文字方向を選択：
    • 縦書き（推奨）: 日本語の美しい縦書き表現
    • 横書き: 横向きのレイアウト
  4. リアルタイムで月文字が生成されます

Step 3: 画像入力の場合

  1. 「画像入力」タブを選択
  2. 画像をドラッグ&ドロップまたは「ファイルを選択」をクリック
  3. JPG、PNG、GIF形式をサポート
  4. 自動で最適化処理され、月文字が生成されます

Step 4: 結果の活用

  • コピー: ワンクリックでクリップボードにコピー
  • テキストダウンロード: .txtファイルとして保存
  • 画像ダウンロード: PNG画像として保存

🎛️ 制御パラメータ

高さ / 幅文字数

  • テキストモード: 10-60行の高さ調整
  • 画像モード: 10-500行の詳細度調整
  • 推奨値: 通常利用では20-40行が最適

フォント / 字體

  • 等幅: 統一された文字間隔、プログラミング風味
  • ゴシック: モダンで読みやすい、汎用性高
  • 明朝: 伝統的で上品、和風表現に最適
  • sans-serif: シンプルで現代的

文字間隔 / 行間調整

  • 文字間隔: -2～+5ピクセルで微調整可能
  • 用途別推奨:
    • SNS投稿: 0～+1
    • 印刷用途: +1～+2
    • モバイル表示: -1～0

余白 / マージン設定

  • 画像モード専用: 0-50ピクセルの枠線追加
  • 効果: アート作品としての完成度向上
  • 推奨値: 10-20ピクセル

反色 / カラー反転

  • 効果: 明暗を逆転、印象的な表現
  • 用途: 暗い背景での表示、特殊効果
  • 組み合わせ: 他のパラメータとの併用で多彩な表現

❓ よくある質問（Q&A）

Q1: 生成された月文字が正しく表示されない

A: 使用しているフォントや表示環境により月の絵文字の表示が異なる場合があります。推奨環境：

  • モダンブラウザ（Chrome, Firefox, Safari, Edge）
  • Unicodeフォントサポート
  • 絵文字表示対応デバイス

Q2: 生成速度が遅い場合の対処法

A: 以下の調整をお試しください：

  • 高さ設定を下げる（100行以下推奨）
  • ブラウザの他のタブを閉じる
  • 画像サイズを小さくする（1MB以下推奨）
  • デバイスの空きメモリを確保する

Q3: 商用利用は可能ですか？

A: はい、生成された月文字アートは自由にご利用いただけます：

  • 商用・非商用問わず利用可能
  • SNS投稿、印刷物、デジタル作品等での利用OK
  • クレジット表記は不要（歓迎します）
  • 再配布や改変も自由

Q4: 日本語以外の言語に対応していますか？

A: はい、多言語対応しています：

  • 英語、中国語（簡体字・繁体字）、韓国語
  • ヨーロッパ言語（フランス語、ドイツ語等）
  • 記号や数字も変換可能
  • 右から左に書く言語は調整が必要な場合あり

Q5: モバイルデバイスでも利用できますか？

A: はい、モバイルフレンドリー設計です：

  • iPhone、Androidスマートフォン対応
  • タブレット端末最適化済み
  • タッチ操作サポート
  • レスポンシブデザイン採用

Q6: データのプライバシーはどうなっていますか？

A: プライバシーを重視しています：

  • アップロードされた画像はブラウザ内のみで処理
  • サーバーへのデータ送信なし
  • 個人情報の収集・保存なし
  • 完全にローカル環境での動作

Q7: 生成結果をSNSで共有する際のコツは？

A: SNSでの効果的な共有方法：

  • Twitter: 文字数制限に注意、画像として投稿推奨
  • Instagram: ストーリーズでの利用が効果的
  • Line: そのままコピペで美しく表示
  • Discord: コードブロック（```）内に貼り付け推奨

Q8: カスタムフォントを使用できますか？

A: 現在、事前に用意された4つのフォントから選択できます：

  • より多くのフォント対応を検討中
  • フィードバックにより追加フォントを決定
  • 特殊フォントリクエストは公式チャンネルまで

Q9: アニメーション効果は追加予定ですか？
